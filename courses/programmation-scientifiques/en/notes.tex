\chapter*{Preface}
\addcontentsline{toc}{chapter}{Preface}

Programming has become an indispensable tool for every scientist. Whether you study mathematics, physics, chemistry, or biology, being able to write a program allows you to automate calculations, visualize data, simulate phenomena, and test hypotheses.

\medskip
These notes are designed for undergraduate students with \textbf{no prior programming experience}. We start from scratch and progress step by step, illustrating each concept with examples from the sciences. The goal is to make you autonomous in solving scientific problems with Python (and an introduction to R).

\medskip
\textbf{Prerequisites.} None in programming. Basic high-school mathematics (functions, equations) is helpful but not required.

\medskip
\textbf{Philosophy.} Each chapter follows the same pattern:
\begin{enumerate}
\item Concept explanation with a scientific analogy.
\item Annotated code with its output.
\item Common errors (``Common Pitfalls'').
\item Scientific mini-project.
\item Graded exercises.
\end{enumerate}

\medskip
\textbf{References.}
\begin{itemize}
\item \textsc{Downey} --- \emph{Think Python}, O'Reilly (freely available online).
\item \textsc{Langtangen} --- \emph{A Primer on Scientific Programming with Python}, Springer.
\item \textsc{VanderPlas} --- \emph{Python Data Science Handbook}, O'Reilly (free online).
\end{itemize}
