Programmation pour Scientifiques

Scientific Programming

Yaé Ulrich Gaba

L1-L2 12 chapitres FR + EN

Description

Ce cours introduit la programmation Python et R pour les scientifiques. De l'installation de l'environnement aux bonnes pratiques de développement, il couvre les bases du langage, les structures de données, et les bibliothèques essentielles du calcul scientifique : NumPy, Matplotlib, Pandas et SciPy.

Table des matières

  1. Installation et Configuration
  2. Introduction à la Programmation
  3. Variables, Types et Opérations de Base
  4. Structures de Contrôle — Conditions et Boucles
  5. Fonctions et Portée
  6. Structures de Données
  7. NumPy — Tableaux et Calcul Numérique
  8. Matplotlib — Visualisation Scientifique
  9. Pandas — Analyse de Données
  10. Introduction à R pour la Statistique
  11. Calcul Scientifique avec SciPy
  12. Bonnes Pratiques — Tests, Documentation, Git

Prérequis

Aucun prérequis en programmation. Notions de mathématiques de niveau L1.
