Bases de Données

Database Systems

Yaé Ulrich Gaba

L2 11 chapters FR + EN

Description

Ce cours présente les fondements des systèmes de bases de données relationnelles et au-delà. On y étudie la modélisation entité-relation, l’algèbre relationnelle, le langage SQL (requêtes, jointures, agrégation), la normalisation, les transactions, l’indexation, les processus ETL et une introduction aux bases NoSQL.
