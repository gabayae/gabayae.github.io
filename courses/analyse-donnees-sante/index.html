Data analysis with Python for health specialists

Analyse de données avec Python pour les professionnels de la santé

Yaé Ulrich Gaba

Graduate 30 hours 10 modules EN + FR

Description

A practical course for health professionals, epidemiologists, and public health students who need to analyze health data with Python. Covers Pandas, descriptive statistics, epidemiological measures, visualization, hypothesis testing, regression, machine learning for clinical prediction, and geospatial health mapping. All datasets are freely available from the WHO, CDC, and UCI repositories.

Table of contents

  1. Module 1 Python essentials for health professionals
  2. Module 2 Health data with Pandas
  3. Module 3 Data cleaning in health contexts
  4. Module 4 Descriptive statistics and epidemiological measures
  5. Module 5 Visualization for health
  6. Module 6 Hypothesis testing
  7. Module 7 Regression for health outcomes
  8. Module 8 Machine learning for clinical prediction
  9. Module 9 Geospatial and temporal health data
  10. Module 10 Capstone project

Prerequisites

No programming experience required. Basic statistics helpful but not mandatory.
