# Data analysis with Python for health specialists

A 30-hour practical course designed for health professionals, epidemiologists, and public health students who need to analyze health data but have little or no programming experience.

## Structure

| Chapter | Topic | Hours | Key datasets |
|---------|-------|-------|-------------|
| 1 | Python essentials for health professionals | 3h | - |
| 2 | Health data with Pandas | 3h | WHO GHO, Gapminder |
| 3 | Data cleaning in health contexts | 3h | NHANES, synthetic EHR |
| 4 | Descriptive statistics and epidemiological measures | 3h | WHO Malaria Report |
| 5 | Visualization for health | 3h | Our World in Data, JHU COVID |
| 6 | Hypothesis testing | 3h | Framingham Heart Study (subset) |
| 7 | Regression for health outcomes | 3h | Pima Indians Diabetes, BRFSS |
| 8 | Machine learning for clinical prediction | 3h | Heart Disease UCI, breast cancer |
| 9 | Geospatial and temporal health data | 3h | Africa CDC, WHO polio |
| 10 | Capstone project | 3h | Student's choice |

## Open datasets used

All datasets in this course are freely available:

- **WHO Global Health Observatory (GHO):** https://www.who.int/data/gho
- **Gapminder:** https://www.gapminder.org/data/
- **Our World in Data:** https://ourworldindata.org/
- **NHANES (CDC):** https://www.cdc.gov/nchs/nhanes/
- **UCI ML Repository:** https://archive.ics.uci.edu/
- **Kaggle open health datasets:** https://www.kaggle.com/datasets
- **Africa CDC data:** https://africacdc.org/resources/
- **JHU COVID-19 data:** https://github.com/CSSEGISandData/COVID-19
- **Framingham Heart Study (teaching subset):** via `statsmodels`
- **World Bank health indicators:** https://data.worldbank.org/

## Prerequisites

No programming experience required. Basic statistics helpful but not mandatory.

## Tools

- Python 3.10+
- Jupyter Notebook / Google Colab
- Libraries: pandas, matplotlib, seaborn, scipy, scikit-learn, lifelines, geopandas

## Languages

- `en/` — English lecture notes (LaTeX + PDF)
- `fr/` — French lecture notes (LaTeX + PDF)
- `code/python/` — Python scripts per chapter
- `code/notebooks/` — Jupyter notebooks per chapter (bilingual)
