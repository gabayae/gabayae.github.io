**The Shape of Data: Geometry-Based Machine Learning and Data Analysis in R** is a book co-authored with [Colleen M. Farrelly](https://scholar.google.com/citations?user=example), published by [No Starch Press](https://nostarch.com/shapeofdata).

### About the Book

This book provides a comprehensive introduction to **geometry-based approaches** in machine learning and data analysis, including:

- **Topological Data Analysis (TDA):** Persistent homology, Mapper algorithm, Betti numbers
- **Geometric methods:** Manifold learning, curvature-based features
- **Practical R implementations:** Hands-on code for real-world datasets
